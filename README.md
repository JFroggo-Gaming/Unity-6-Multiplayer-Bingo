# Unity 6 Multiplayer Bingo

A multiplayer Bingo game built with Unity 6 and Mirror Networking.

## About

This project implements a networked Bingo game using Unity 6 and the Mirror networking framework. Players can connect to a server and play Bingo together in real-time.

## Technical Stack

- **Unity Version**: 6000.0.59f2
- **Networking**: Mirror Networking
- **Platform**: Multi-platform support

## Features

- Real-time multiplayer gameplay
- Mirror networking integration
- Cross-platform compatibility

## Getting Started

### Prerequisites

- Unity 6000.0.59f2 or later
- Basic understanding of Unity and networking concepts

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/Unity-6-Multiplayer-Bingo.git
   ```

2. Open the project in Unity Hub

3. Open the main scene and press Play to start

## Project Structure

- `/Assets` - Main project assets and scripts
- `/Packages` - Unity packages and dependencies
- `/ProjectSettings` - Unity project configuration
- `/BingoBuilds` - Compiled game builds

## Mirror Networking

This project uses [Mirror Networking](https://mirror-networking.com/) for multiplayer functionality. Mirror is a high-level networking library for Unity that makes it easy to create multiplayer games.

## Building

To create a build:

1. Go to File > Build Settings
2. Select your target platform
3. Click Build or Build and Run

Builds will be output to the `BingoBuilds/` directory.
